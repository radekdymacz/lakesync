## Installation

```bash
npm install lakesync
```

Or with other package managers:

```bash
pnpm add lakesync
# or
bun add lakesync
```

## Quick Start

### 1. Create a Local Database

```ts
import { LocalDB } from "lakesync";

const db = await LocalDB.create();
```

### 2. Set Up a Sync Tracker

The `SyncTracker` wraps your local database, a queue, and a hybrid logical clock. It automatically extracts deltas from every insert, update, and delete operation.

```ts
import { SyncTracker, MemoryQueue } from "lakesync";
import { HLC } from "lakesync";

const hlc = new HLC();
const queue = new MemoryQueue();
const tracker = new SyncTracker(db, queue, hlc, "client-1");
```

### 3. Define a Table

```ts
db.exec(`
  CREATE TABLE IF NOT EXISTS todos (
    id TEXT PRIMARY KEY,
    title TEXT NOT NULL,
    completed INTEGER NOT NULL DEFAULT 0
  )
`);
```

### 4. Insert Data

```ts
tracker.insert("todos", {
  id: "todo-1",
  title: "Build something great",
  completed: 0,
});
```

### 5. Connect a Transport and Sync

For local development, use `LocalTransport` to sync in-process:

```ts
import { SyncCoordinator, LocalTransport } from "lakesync";
import { Gateway } from "@lakesync/gateway";

const gateway = new Gateway("gw-1");
const transport = new LocalTransport(gateway, "client-1");

const coordinator = new SyncCoordinator({
  tracker,
  transport,
  syncMode: "manual",
});

// Push local changes to the gateway
await coordinator.push();

// Pull remote changes from the gateway
await coordinator.pull();
```

For production, use `HttpTransport` to connect to a remote gateway:

```ts
import { HttpTransport } from "lakesync";

const transport = new HttpTransport({
  baseUrl: "https://your-gateway.example.com",
  gatewayId: "gw-1",
  getToken: async () => "your-jwt-token",
});
```
